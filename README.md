## Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [android-test-automation](#android-test-automation) | Android UI test automation agent that executes tasks on real devices and crystallizes them into pytest scripts | 0.1.0 |

---

## Installation

### Add this marketplace

Register this repository as a Claude Code marketplace:

```bash
claude plugin marketplace add lanbaoshen/awesome-claude-code
```

### Install a plugin

```bash
claude plugin install <plugin-name>@awesome-claude-code
```

For example:

```bash
claude plugin install android-test-automation@awesome-claude-code
```
